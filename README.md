# Notes
Notes is a tiny notepad for almost any 32-bit/64-bit Windows version.
It will run on anything from Windows 11 to Windows 98SE (I haven't tested either of those yet but basic logic says it should)

## Dependencies
- Just COMDLG32.OCX!

## Features
- Fonts
- Right-aligned (named Right to Left but the Text object doesn't allow proper RTL)
- and under 32kb!

## Missing (for now)
- Word wrap

## Keep in mind
- This works only on systems with the VB6 runtime installed. Any OS 98SE+ should have it pre-installed.
- The installer was built with CreateInstall and might not work on anything before XP. The binary itself should work fine though!
- This app was not vibe-coded.
